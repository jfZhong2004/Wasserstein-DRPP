# 一维 W-DRPP 实验代码使用说明

## 1. 简要介绍

本项目实现了系统 C 下的一维 W-DRPP 实验流程（实验 2 主线），包含：

- Zero / PID 两种控制策略
- Nominal、Noise-DRPP、KDE、Oracle、W-DRPP 上下界
- 固定步长 `k=0..31`（共 32 步）
- Monte Carlo 评估与逐步 log-score 可视化

其中 W-DRPP 精确求解器复用：

- `src/solvers/drpp_1d_exact_solver.py`

## 2. 代码结构（核心）

- `src/runner/exp2_systemC.py`：实验入口
- `src/data/build_dataset.py`：历史数据集生成与读取
- `src/controllers/`：Zero/PID 控制与 PID 自动整定
- `src/noise_lib/noise_w1_w6.py`：W1~W6 噪声分布（含 W4 截断+标准化）
- `src/predictors/`：各预测器实现
- `src/radius/wasserstein_radius.py`：Theorem 3.4 / Eq.(8) 半径计算
- `src/viz/plot_per_step_scores.py`：结果曲线绘图

## 3. 运行前准备

建议 Python 3.10+，安装依赖：

```bash
pip install numpy scipy matplotlib
```

## 4. 运行方式

在工作目录执行：

```bash
python -m src.runner.exp2_systemC --m 1000 --steps 32 --n-main 100
```

常用参数：

- `--m`：Monte Carlo 次数
- `--steps`：预测步数（当前固定应为 32）
- `--n-main`：实验 2 主样本量
- `--beta --a --c1 --c2`：Wasserstein 半径参数
- `--dataset-seed-master`：训练数据集主种子
- `--eval-seed-master`：评估轨迹主种子（应与训练不同）
- `--datasets-root`：数据集目录
- `--results-root`：结果目录
- `--skip-dataset-build`：跳过数据集重建

图像纵轴默认使用稳健分位数范围（2%~98%，带少量边距），
以避免个别极端点压缩其他曲线可读性；超出范围的点会被裁出可视区。

## 5. 输出结果

运行后将生成：

1. 数据集目录（默认）`datasets_1d_wdrpp/`
2. 时间戳结果目录（默认）`results_1d_wdrpp/run_YYYYMMDD_HHMMSS/`

结果目录主要文件：

- `experiment_config.yaml`：本次实验配置
- `requirement_trace.md`：需求追踪
- `per_step_scores.csv`：每步各方法均值/方差
- `summary_table.csv`：整体汇总分数
- `figures/*.png`：每个噪声与控制配置的曲线图

## 6. 复现实验的关键固定项

- 初始状态：`x0 = 2.0`
- 训练主随机种子：`dataset_seed_master = 20260412`
- 评估主随机种子：`eval_seed_master = 30260412`（与训练不同）
- 控制限幅：`u in [-4, 4]`
- 置信参数：`beta = 0.05`
- 评估指标：仅 log-score
